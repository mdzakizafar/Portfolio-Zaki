## 🚀 How to Run the Project Locally

### 1. Clone the repository

```bash
git clone https://github.com/mdzakizafar/Portfolio-Zaki.git
```

### 2. Open the project directory

```bash
cd Portfolio-Zaki
```

### 3. Open the project in Visual Studio Code

```bash
code .
```

### 4. Run the website

Open the `index.html` file directly in a web browser.

For a better development experience, install the **Live Server** extension in Visual Studio Code.

Then:

1. Right-click on `index.html`.
2. Select **Open with Live Server**.
3. The portfolio will open in your default web browser.
